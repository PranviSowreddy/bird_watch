# 🦜 Indian Bird Identifier

A Streamlit-powered AI application that identifies **25 common Indian bird species** from photos — complete with conservation status, GBIF habitat maps, Wikipedia summaries, and fun facts.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🔍 **Top-3 Predictions** | Confidence bars + medal rankings for each prediction |
| 🏅 **IUCN Status Badge** | Color-coded conservation status (LC / NT / VU / EN / CR) |
| 🗺️ **Interactive Habitat Map** | Real GBIF occurrence points plotted on a Folium map |
| 📊 **Sightings Timeline** | Bar chart of GBIF sightings by year |
| 📚 **Wikipedia Summary** | Auto-fetched first paragraph (cached) |
| 💡 **Fun Facts** | Curated facts for all 25 species |
| 🌿 **Species Grid** | Browse all 25 species with IUCN badges before uploading |

---

## 🏗️ Architecture

```
app.py              ← Streamlit UI (single-file)
model_utils.py      ← EfficientNet-B0 inference + CLIP fallback
api_utils.py        ← Wikipedia + GBIF + Folium map builder
train_model.py      ← Fine-tuning script (run once)
cache/
  species_data.json ← IUCN status, fun facts, GBIF keys (pre-filled)
  wiki_cache.json   ← Wikipedia summaries (auto-populated on first run)
bird_classifier.pth ← Trained weights (generated by train_model.py)
```

### Model Strategy

| Priority | Model | When Used |
|---|---|---|
| 1st | **EfficientNet-B0** (fine-tuned) | When `bird_classifier.pth` exists |
| Fallback | **CLIP** zero-shot | When no trained weights found |

## 🚀 Quick Start

### Option A — With trained model (recommended)

```bash
# 1. Clone / download this folder
cd bird_app

# 2. Install dependencies
pip install -r requirements.txt

# 3. Set up Kaggle credentials (one time)
# Go to https://www.kaggle.com/settings → API → Create New Token
# Move kaggle.json to ~/.kaggle/kaggle.json

# 4. Download dataset + train model (20-60 min CPU / ~5 min GPU)
bash setup.sh --with-dataset

# 5. Launch the app
streamlit run app.py
```

### Option B — CLIP zero-shot (no training needed)

```bash
pip install -r requirements.txt
streamlit run app.py
# The app auto-uses CLIP if bird_classifier.pth is absent
```

## 🤖 Model Details

### EfficientNet-B0 Fine-tuning
- **Base:** `timm/efficientnet_b0` (ImageNet pretrained)
- **Epochs:** 20 with CosineAnnealingLR
- **Optimizer:** AdamW (lr=1e-4, weight_decay=1e-4)
- **Augmentation:** RandomResizedCrop, ColorJitter, RandomRotation, HorizontalFlip
- **Loss:** CrossEntropyLoss with label smoothing (0.1)
- **Expected val accuracy:** ~90–95% after 20 epochs

### CLIP Zero-Shot
- **Model:** `openai/clip-vit-base-patch32`
- **Prompts:** `"a photo of a {species}, an Indian bird"`
- **No training required**

## 🔧 Configuration

All species metadata (IUCN status, fun facts, GBIF taxon keys, rarity) is stored in `cache/species_data.json` and can be edited directly without retraining.

To add a **Gemini API** key for AI-generated fun facts (optional):
```bash
export GOOGLE_API_KEY="your-key-here"
```
The app uses the pre-filled `species_data.json` by default (no key required).
